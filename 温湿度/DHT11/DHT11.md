大叔教你玩板子–外设系列（一）温湿度-SHT20

### 1.传感器介绍

 SHT20数字温湿度传感器是SHT21温湿度传感器系列中一款性价比高的产品，应用领域广泛。该类传感器适用于对成本极其敏感但又注重品质的大批量生产的行业。完全校准、数字输出，I 2 C接口、低功耗、优良的长期稳定性等特点

### 2.硬件连接

|   ESP32    | SHT20 |
| :--------: | :---: |
|    VCC     |  VCC  |
|    GND     |  GND  |
| P19/GPIO22 |  SCL  |
| P20/GPIO21 |  SDA  |
|            |       |
